About AWS Speed Test

AWSSpeedTest.com is a speed test tool that measures your network latency across multiple global locations. It is independently maintained by Blair Chen and the GitHub community.

About this project

This site is not officially associated or endorsed by Amazon, and all information is based on publicly available AWS documentation and resources. The source code is accessible on GitHub repository. We believe in open-source development and welcome community contributions to improve this tool.

Feedback & contributions

If you come across any bugs or have a feature suggestion, kindly create an issue on GitHub Issues.

We welcome and greatly appreciate your contributions. If you're interested in contributing, please fork the repository and submit a pull request.
