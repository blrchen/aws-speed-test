About AWS Speed Test

AWSSpeedTest.com is a speed test tool that measures your network latency across multiple global locations. It is independently maintained by Blair Chen and the Github community in spare time, this site is not officially associated or endorsed by Amazon. The source code is accessible on GitHub . This project is built using various open-source technologies such as:

  • Angular
  • Bootstrap
  • AWS Catalog
  • d3-shape
  • ngx-charts
  • RxJS
