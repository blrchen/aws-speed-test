About AWS Speed Test

AWSSpeedTest.com is a speed test tool that measures your network latency across multiple global locations. It is independently maintained by Blair Chen and the GitHub community in spare time. This site is not officially associated or endorsed by Amazon, and all information is based on publicly available AWS documentation and resources. The source code is accessible on GitHub .
