What does geography mean in this directory?

A geography groups AWS regions by the market or country label in the current infrastructure dataset. One geography can contain one or more regions, and each region contains multiple isolated Availability Zones.

Treat this grouping as a navigation aid. AWS service documentation, compliance programs, and service-specific data residency guidance are the authoritative sources for workload decisions.

AWS geography vs. region

These terms describe different levels of the AWS infrastructure directory.

Geography
A market or country grouping containing one or more AWS regions.
Region
A separate geographic area identified by a stable code such as us-east-1.
Availability Zone
An isolated location within a region, connected through low-latency networking.
Region group
A broader directory grouping such as Asia Pacific, Europe, or North America.
