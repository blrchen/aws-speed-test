About AWS Speed Test

AWSSpeedTest.com is an AWS speed test tool that allows you to test your network latency to AWS datacenters around the world.

Blair Chen maintains AWSSpeedTest.com as an open-source project on GitHub . This project utilizes various open-source technologies, including:

  • Angular
  • Bootstrap
  • Cloud Catalog
  • d3-shape
  • ngx-charts
  • RxJS
