About

www.AWSSpeedTest.com - AWS speed test tool. Test your network latency to AWS datacenters around the world.

www.AWSSpeedTest.com is maintained by Blair Chen(@blrchen), open source on GitHub, built on top of following open source projects
  • Angular
  • Application Insights JavaScript SDK
  • ASP.NET Core
  • Cloud Data Lab
  • Bootstrap
  • d3-shape
  • Font Awesome
  • ngx-charts
  • RxJS
